# GitStats Pro - GitHub Badge Generator

A professional tool to create and customize GitHub statistics badges for your profile and projects.

**Created by KaiTooast**

## Features

- **GitHub User Info**: Automatically fetch information from GitHub users
- **Multiple Badge Designs**: Choose from several predefined themes
- **Live Preview**: See your badges in real-time
- **SVG Download**: Download your badges as SVG files
- **Responsive Design**: Works on all devices
- **Custom Colors**: Create your own color schemes

5. Download the SVG files or copy the Markdown code

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for GitHub API requests

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub REST API

## How It Works

1. Enter your GitHub username
2. The app fetches your public GitHub data via the GitHub API
3. Generates custom SVG badges based on your selected theme
4. Download the SVGs to use in your GitHub README or project documentation

## Installation

Simply download the files and open `index.html` in your browser. No installation or build process required!

## Usage

### For GitHub README

1. Download the SVG badges
2. Create a `badges` folder in your GitHub repository (optional)
3. Upload the SVG files to your repository
4. Add the following to your README:

\`\`\`markdown
![GitHub Stats](./badges/stats.svg)
![Languages](./badges/languages.svg)
![Streak](./badges/streak.svg)
\`\`\`

## Tips

- For best results, use SVG downloads directly in your repositories
- Custom color picker allows for perfect theme matching
- All data is fetched from public GitHub APIs
- No registration or API keys required
